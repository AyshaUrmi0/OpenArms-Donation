What is the Document Object Model ?

DOM is a tree-like structure that represents all the elements like buttons, paragraphs, images in the HTML. It allows JavaScript to interact with and modify the content, structure, and style of a web page.

How do you select an element from DOM?

Using various js methods we can select an element from DOM. Like, querySelector(), querySelectorAll(),getElementById, getElementsByClassName, etc.

what is event deligation in context of dom and why it is useful?

Event deligation is the mechanism by which an event is passed from one element to another. It helps in event bubbling. It happens when an element receives an event, and that event bubbles up to its parent

How do you manipulate an elements attributes and style using the DOM?

Using various js methods we can manipulate an elements attributes and style using the DOM. Like, setAttribute(), getAttribute(), setStyle(), etc.
